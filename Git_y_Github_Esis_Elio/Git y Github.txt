a. ¿Quién inventó el sistema de control de versión Git y por qué?

El sistema de control de versiones Git lo desarrollo Linus Torvalds en el año 2005, fue diseñado para poder conserver la integridad del codigo fuente gestionado, permitiendo flexibilidad y control de versionado para grandes grupos y flujos de trabajo de desarrollo no lineal, aumentando la eficiencia.

b. ¿A quién pertenece actualmente Github y por qué?

Actualmente le pertenece a la empresa Microsoft, lo compra a parte del potencial de la herramienta, para reforzar el programa, ya que ese año habria sufrido ataque DDos, y de esta forma mostraron compromiso para mantener la libertad e innovacion de los desarrolladores
