a. ¿Quién inventó el sistema de control de versión Git y por qué?
Git es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.

b. ¿A quién pertenece actualmente Github y por qué?
El 4 de junio de 2018, Microsoft compró GitHub. Desde el nombramiento de Satya Nadella como CEO de Microsoft , la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios.
