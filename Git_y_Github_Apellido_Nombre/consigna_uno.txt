a. ¿Quién inventó el sistema de control de versión Git y por qué?
    El sistema de control de versiones Git fue creado por Linus Trovadls creador de Linux, para facilar la 
    coordinación del trabajo de múltiples personas sobre archivos compartidos en un mismo repositorio.
