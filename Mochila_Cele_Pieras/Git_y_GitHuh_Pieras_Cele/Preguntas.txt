a.	GIT es un software de control de versiones diseñado por Linus Torvalds para controlar el desarrollo del kernel de Linux, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente.
b.	GITHUB pertenece actualmente a Microsoft porque fue adquirida por ellos el 4 de junio de 2018, por un importe de 7500 millones de dólares. Entre los motivos se destaca que desde el nombramiento de Satya Nadella como CEO de Microsoft, la compañía ha adoptado una estrategia nueva, mucho más abierta al mundo del software libre y a la integración con otras plataformas y servicios. Además, hay otro motivo de peso: Microsoft Azure. En plena explosión de cloud services (servicios en la nube), con un crecimiento previsto para este año del 21.4%, Microsoft tiene que competir con el líder indiscutible hoy, la todopoderosa Amazon con AWS. Qué mejor forma de hacerlo que estar en contacto directo con los desarrolladores que trabajan en esos servicios, ver sus necesidades y ofrecerles herramientas que faciliten su integración con Azure.
c.	Además utilizar GITHUB desde la terminal, se puede utilizar desde el sitio web directamente y también desde las aplicaciones de GITHUB para Celulares
